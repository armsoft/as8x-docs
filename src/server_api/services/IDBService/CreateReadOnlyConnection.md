```c#
public SqlConnection CreateReadOnlyConnection(bool pooling = true);
```

Ստեղծում է միայն կարդալու իրավասությամբ լրացուցիչ SQL միացում դեպի հիմնական տվյալների պահոց։

**Պարամետրեր**

* `pooling` -  Օգտագործել միացումների քեշ, թե ոչ։
